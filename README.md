## Development

## Vue Configuration

See [Configuration Reference](https://cli.vuejs.org/config/).

### Yarn Commands

#### Install Dependencies

```
yarn install
```

#### Compile and Serve with Hot-Reloads Development

```
yarn serve
```

#### Lints and fixes files

```
yarn lint
```

#### Compile and Minify for Production

```
yarn build
```

## Deployment

1. Run `yarn lint`.
2. Run `yarn build`.
3. Run `git add .`.
4. Run `git commit -m `
